Participation

  1. In order to participate, you will need to create a Fifth Aeon account and create or join a team.
  2. A team may have between 1 and 4 individuls.
  3. You may not participate in more than one team or use more than one account in a single tournament.
  4. In order to participate you will need to fill out a reaserch survey before and after the tournament.
  5. If you win a prize you will be required to write a 2 page description of your bot.

Game Mechanics

  1. An A.I will be given a single process on the competition server to run on
  2. Each time an A.I gains priority it will have 5 seconds to make computations and send actions. If it fails to pass priority in that time, it will lose the game
  3. If an A.I sends an illegal action to the server, or an action which causes the game process to throw an exception it will lose the game

Banned Behaviors

  1. Bots may no deliberately lose games.
  2. Bots are not allowed to access external resources such as remote servers or create additional processes.
  3. Bots may not access any information that would not be available to a human player of the game. However, they may remember previously seen information or make inferences.
